# How to Open the Inventory App

This app does not need to be installed. It runs directly in a web browser.

## Step 1: Download the app

Download the `inventory.zip` file to your computer.

## Step 2: Extract the ZIP file

### Windows

1. Right-click `inventory.zip`.
2. Select **Extract All**.
3. Click **Extract**.

### Mac

Double-click `inventory.zip`. Your Mac will automatically create an extracted inventory folder.

## Step 3: Open the app

1. Open the extracted `inventory` folder.
2. Find the file named `index.html`.
3. Double-click `index.html`.
4. The inventory app will open in Chrome, Edge, Safari, Firefox, or your default browser.

## Important

Keep these items together in the same folder:

- `index.html`
- `style.css`
- `script.js`
- The `assets` folder

Do not move `index.html` away from the other files, or the app's design and background images may not load.

## Saving inventory

The app automatically saves inventory inside the web browser. There is no Save button.

- Use the same computer and browser to see previously saved inventory.
- Clearing browser data may delete saved inventory.
- Different computers and browsers have separate inventory data.
- Use **Export CSV** to create a backup of the inventory.

## Troubleshooting

If double-clicking `index.html` does not open the app:

1. Right-click `index.html`.
2. Choose **Open with**.
3. Select Chrome, Edge, Firefox, or Safari.

If the page does not look correct, confirm that `style.css`, `script.js`, and the `assets` folder are still beside `index.html`.
